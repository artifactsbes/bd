# Orchestration, Interoperability, and Architecture in LLM-Based Multi-Agent Systems: A Systematic Mapping Study

This repository contains the anonymized methodological artifact used in the systematic mapping study on the integration between Multi-Agent Systems (MAS) and Large Language Models (LLMs).

The spreadsheet includes the organization of the search results, study selection records, deduplication control, classification of studies by research question, evidence extraction protocol, coding matrix, and synthesis of category incidence for each RQ.

The artifact supports the traceability of the mapping process and provides the data used to summarize tools, orchestration mechanisms, communication strategies, interoperability approaches, architectures, applications, challenges, and research gaps identified in the analyzed literature.
